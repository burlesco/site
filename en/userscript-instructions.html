Instructions to add the userscript

Userscript is the Burlesco version to be used with TamperMonkey. It works on CHrome, Edge, Opera and Firefox. Every site supported by extension also works on userscript.

Here are the instructions:

  1. Install the Tampermonkey extension
  2. Add the script burlesco.user.js
  3. Clean your browser cache (instructions)
  4. Enjoy Burlesco!

That's the preferred method for use with Chrome, since the manual installation doesn't have automatic updates.
